# MedSpa Showcase Custom Theme (2026)

## Overview
This theme was developed from scratch to demonstrate senior-level WordPress architecture, utilizing a clean PHP structure and modern front-end standards. 

## Technical Context
Following a hardware loss incident during Typhoon Odette, this project serves as a fresh demonstration of my current architectural standards, moving away from legacy templates toward high-performance, custom-built solutions.

## Architecture
- Custom Post Types for Service Management
- Optimized PHP Logic (no bloated plugins)
- Tailwind CSS / Modern CSS Variables

## Deployment & Security
- Developed locally using LocalWP for a clean environment.
- Hardened against common vulnerabilities by removing WordPress versioning.
- Managed via Git with a strict .gitignore to prevent configuration leaks.
- Rebuilt as a modern demonstration of resilience and technical standard following hardware 
  loss in 2021.
